# 🤖 Agentes Inteligentes para Análise de Dados e Análise de Currículos

Este repositório contém dois prompts profissionais criados para gerar agentes de IA altamente eficazes, prontos para uso em plataformas como:

- ChatGPT  
- M365 Copilot  
- Claude  
- LLMs customizados  

Os agentes foram desenhados para entregar resultados reais em cenários práticos:
- **Análise de Dados com upload de arquivos**
- **Análise de Currículos com base em uma vaga**

# 💬 Exemplos de Interação

---

## 📊 Exemplo — Agente de Análise de Dados

**Usuário:**  
Bom dia, quero analisar o faturamento da empresa.

**Agente:**  
Bom dia! ☀️  
Claro! Para começarmos, por favor envie seu arquivo Excel, CSV, JSON ou PDF.  
Assim que eu receber, preparo a análise para você.  
Qual será exatamente sua consulta hoje? 😊

---

## 📄 Exemplo — Agente de Currículos

**Usuário:**  
Boa tarde. Quero saber se meu currículo combina com esta vaga (PDF anexado).

**Agente:**  
Boa tarde! 🌤️  
Perfeito! Já recebi a vaga.  
Agora, por favor envie seu currículo (PDF, Word ou outro formato) para que eu possa fazer a análise completa.
